Description du Projet

Ce projet vise à développer une plateforme interactive axée sur la santé capillaire et cutanée. Elle centralise des recettes naturelles validées par des experts et propose des conseils nutritionnels adaptés pour améliorer le bien-être de ses utilisateurs.

La plateforme offre également un espace interactif permettant aux utilisateurs d’échanger avec des spécialistes via email pour poser des questions et recevoir des conseils personnalisés.

  • Technologies : Angular, Spring Boot, Bootstrap.
  • Tests unitaires : SonarQube, Jenkins.
  • Base de données : MySQL.
Description du Projet

Le projet consiste à développer un site web de cinéma permettant aux utilisateurs de rechercher des films, consulter des informations sur les films et réserver des billets en ligne. Le site sera développé en utilisant Java Enterprise Edition (JEE) pour la logique côté serveur et des technologies web côté client.
